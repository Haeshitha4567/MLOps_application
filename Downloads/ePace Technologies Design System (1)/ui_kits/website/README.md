# ePace Technologies — Website UI Kit

High-fidelity recreation of the `epacetech.com` marketing site. Components are React + Babel-in-browser — simple, mostly-cosmetic, designed to be pieced together.

## Components

| File | Purpose |
|---|---|
| `Chrome.jsx` | `TopBar`, `Header`, `Footer` — site chrome |
| `Hero.jsx` | Hero section with split layout + glass stats card |
| `StatsStrip.jsx` | Dark 4-column stat counters |
| `Services.jsx` | Services grid (9 cards) + section title |
| `AISection.jsx` | Dark AI feature section with capability bars |
| `GlobalSection.jsx` | Global delivery split layout with location panel |
| `Process.jsx` | Dark 4-step process |
| `Clients.jsx` | Logo marquee |
| `Testimonials.jsx` | 3-up testimonial grid |
| `CTA.jsx` | Dark hero-gradient call-to-action |
| `ContactSection.jsx` | Split contact info + form card |
| `PageHeader.jsx` | Inner page header with breadcrumb |

## Screens covered by the click-thru (`index.html`)

1. **Home** — full homepage composition
2. **Services** — services detail page
3. **Careers** — open positions list with badges
4. **Contact** — contact form and info
