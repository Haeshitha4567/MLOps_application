# Startup Growth Intelligence Engine

An end-to-end Machine Learning Operations (MLOps) platform designed to solve two of the most critical challenges for modern SaaS companies: **User Retention (Churn)** and **Customer Feedback Analysis**.

This repository demonstrates the full ML lifecycle: from synthetic data generation and model training to API deployment and a premium interactive frontend dashboard.

## 🚀 Features

1. **Customer Churn Predictor (Random Forest)**
   - Trains a `scikit-learn` Random Forest Classifier on SaaS customer metrics (Days Since Login, Support Tickets, Monthly Spend, Feature Adoption).
   - Serves real-time predictions via a robust API, returning churn probability and dynamic risk categorization (Critical, High, Moderate, Low).

2. **Feedback Sentiment Analyzer (NLP)**
   - Utilizes a Natural Language Processing (NLP) pipeline combining a `TfidfVectorizer` and a `LogisticRegression` model.
   - Automatically classifies unstructured user feedback as Positive, Negative, or Feature Request to help product teams prioritize updates.

3. **FastAPI Inference Server**
   - High-performance asynchronous Python web server that loads serialized (`.pkl`) models into memory.
   - Provides clean REST API endpoints (`/api/predict/churn` and `/api/predict/sentiment`) for seamless frontend integration.

4. **Premium "Glassmorphism" Dashboard**
   - A stunning, zero-dependency (Vanilla HTML/CSS/JS) frontend UI built with a modern dark-mode glass aesthetic.
   - Features animated confidence bars, dynamic risk badges, and interactive sliders to simulate real-time model inference.

## 🛠 Tech Stack

- **Machine Learning**: `scikit-learn`, `pandas`, `numpy`, `joblib`
- **Backend API**: `FastAPI`, `uvicorn`, `pydantic`
- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism), JavaScript (ES6+), FontAwesome
- **Architecture**: Decoupled Client-Server MLOps Architecture

## 💻 How to Run Locally

### 1. Setup the Python Backend
Ensure you have Python 3.9+ installed.

```bash
# Clone the repository
git clone https://github.com/yourusername/growth-intelligence-engine.git
cd growth-intelligence-engine

# Install dependencies
pip install -r backend/requirements.txt

# (Optional) Train the models from scratch (pre-trained models may be included)
python backend/train_models.py

# Start the FastAPI server
python -m uvicorn backend.main:app --host 127.0.0.1 --port 8000
```
*The backend should now be running at `http://127.0.0.1:8000`.*

### 2. Launch the Frontend
You can run the frontend using any local HTTP static server. For example:

```bash
# In a new terminal tab, navigate to the frontend folder
cd frontend

# Start a static file server
python -m http.server 3000 --bind 127.0.0.1
```
*Navigate your web browser to `http://127.0.0.1:3000/index.html` to access the dashboard.*
